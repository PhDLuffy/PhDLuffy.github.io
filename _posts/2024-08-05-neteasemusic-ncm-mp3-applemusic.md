[转换网易云音乐](https://github.com/taurusxin/ncmdump)

Mac下安装

```bash
brew install ncmdump
```

网易云音乐下载歌曲→在finder中显示

转换命令

```bash
ncmdump file-path
```

同文件夹下生成320k的mp3
